# ADR-0001：将 10×RTX 3090 服务器设为主平台

## 状态

Accepted

## 背景

项目可使用：

- 10×RTX 3090 24GB。
- 8×V100 32GB。

RTX 3090 单卡显存较小，但总卡数更多，并支持 BF16 和更新的软件生态。

## 决策

- RTX 3090 服务器作为默认开发、训练和推理平台。
- V100 服务器作为兼容性和对照平台。
- 标准训练组使用 8 张 RTX 3090。
- 剩余 2 张用于评测、推理、开发和备用。

## 原因

- 更适合 BF16。
- 更适配现代框架。
- 10 卡可支持 FSDP2/ZeRO-3。
- 可分离训练和评测资源。
- 便于形成可复现的 1/2/4 卡扩展报告，并在资源允许时追加 8 卡证据。

## 后果

- 必须先完成拓扑和 NCCL 测试。
- 不能把 10×24GB 简化成单块 240GB 显存。
- DDP 不用于解决单卡放不下模型的问题。
- 7B Full SFT 需要 FSDP2 或 ZeRO-3。
